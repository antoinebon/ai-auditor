# ai-auditor

ISO 27001:2022 policy gap analyzer. Reads a security policy PDF, checks it against
a curated set of Annex A controls, and produces a structured gap analysis report
with cited evidence per control.

Built with LangGraph and a local Ollama model — everything runs on your machine.
